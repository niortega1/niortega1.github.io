ABOUT

Hi there! I'm Nicolas Ortega, a Front-End Engineer from Indianapolis with a knack for creating elegant web solutions. With a background in Kinesiology from Indiana University and a sharp turn into the tech world, I've found my niche in melding user-centric design with robust front-end development. Armed with tools like HTML/CSS, JavaScript, React, and a suite of other modern technologies, I craft intuitive applications that prioritize both form and function.

From leading the design of a Medical Dashboard to developing the full-stack features of DeviceDen, my work reflects a commitment to excellence and a passion for solving challenges through code. If you're on the lookout for someone who blends a scientific approach with creative coding to elevate user experiences, let's connect and bring your next project to life!

Work And Projects

Medical Dashboard Project

GitHub | Demo

This Medical Dashboard is a bespoke interface developed to emulate the daily operations of medical professionals and administrators. It offers a fully functional dashboard to visualize and manage patient information, scheduling, billing, and much more.



G&C Painting and Repair | Client Website

Demo

Boca Premier Painting Services is a professional painting service provider located in the heart of Boca Raton, FL. Our website serves as a digital front door to our business, offering clients the opportunity to explore our painting services, view past projects, and request free estimates and quotes directly through an intuitive contact interface.



DeviceDen Project

GitHub | Demo

DeviceDen is a mock e-commerce platform designed to showcase a streamlined shopping experience with full integration of Stripe Payment Processing. This project serves as a demonstration of a functional online store where users can explore products and simulate transactions.



Jammming Project

GitHub | Demo

Jammming is a single-page application that allows users to search the Spotify library, create a custom playlist, and then save it to their Spotify account. It offers a seamless music curation experience by integrating directly with Spotify's API.



SnapShot Project

GitHub | Demo

SnapShot is a static website concept designed to act as a landing page for a photo/image sharing platform. This project aims to showcase design skills and the thought process behind creating a user-friendly interface for a service that offers free image downloads.
